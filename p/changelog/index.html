Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

[0.4.0] - 2025-10-15

Added

  • Implemented a quick tag popover that appears after pinning a chat.
  • Added an option in the General tab of the settings modal to disable the quick tag popover.

Fixed

  • Addressed a bug where the text in expanded overlay was not clickable.

[0.3.0] - 2025-10-08

Added

  • Testing infrastructure with unit and E2E tests
  • Focus trap implementation for all modals and popovers
  • Svelte 5 migration with modern UI components
  • Schema versioning system for data persistence
  • Modal service with type-safe management
  • Chat title rename sync across devices
  • Chat delete sync functionality
  • Logging system with contextual levels
  • Extension update notification toast
  • About tab with external resource links

Changed

  • Migrated from legacy UI to Svelte 5 components
  • Improved modal animations using Svelte transitions instead of CSS
  • Enhanced focus management and accessibility across all components
  • Refactored state management with modular architecture
  • Optimized Vite build configuration and ES module handling
  • Improved code organization with better separation of concerns
  • Enhanced error handling and validation throughout the application
  • Optimized bundle size and performance
  • Updated TypeScript configuration for better type safety
  • Enhanced i18n support with smart native language display

Fixed

  • Modal focus bugs and trap behavior
  • Tag popover keyboard navigation and click detection
  • Focus ring visibility for interactive elements
  • ESLint configuration and code quality issues

Security

  • Resolved security vulnerabilities in dependencies

[0.2.0] - 2024-12-15

Added

  • Initial i18n support for multiple languages
  • Dynamic language refresh for side panel
  • JSON validation script for i18n messages
  • Enhanced sidebar button styling
  • Improved modal action buttons

Changed

  • Migrated from legacy build system to Vite with ES modules support
  • Updated project structure for better maintainability
  • Enhanced validation scripts for better error reporting

Fixed

  • Missing pinned chats icon in sidebar
  • Message substitutions in translations
  • Missing translation keys
  • JSON validation issues

[0.1.0] - 2024-11-20

Added

  • Initial release of ChatGPT Pin Chats extension
  • Core pinning functionality for ChatGPT conversations
  • Tag management system with color customization
  • Search functionality for pinned chats with fuzzy matching
  • Collapsible sidebar with toggle functionality
  • Modal system for managing pinned chats
  • Basic styling and animations
  • Chrome extension manifest v3 support
  • Cross-device sync support via Chrome storage
  • Basic keyboard shortcuts for modal operations
  • Color picker for tag customization
  • ESC key support for closing modals

Changed

  • Improved modal positioning and styling
  • Enhanced icon consistency
  • Better drawer animation behavior

Fixed

  • Icon flashing on page load
  • Select options visibility issues
  • Tag assignment popup behavior
  • Modal positioning and styling inconsistencies

Keep your ChatGPT conversations organized and accessible across all devices.
